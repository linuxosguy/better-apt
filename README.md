# Dependencies:
- sudo
- apt
- apt-cache

# Fully Install bapt Guide
To fully install bapt and set it up with for terminal to run anywhere by typing bapt, follow the instructions below.
- Open a shell
- Run command:
```bash
sudo ln -s /path/to/bapt-linux.sh /usr/bin/bapt
```
Note: You need to keep the file to exist, and in the same place, in case of moving, run: 
```bash
sudo rm -rf /usr/bin/bapt
```
and run the link command again.
