# better-apt
Better APT is a wrapper (soon to be it’s own package manager) for the APT package manager.

This is a one-man project, so please don't complain, I'm working really hard to push out releases.
Version v1.0.0, may be buggy and not that much packed with features, this is just the foundation.
You can either stay and support the project while you wait for releases that are packed with features, or stay and help support the project!
